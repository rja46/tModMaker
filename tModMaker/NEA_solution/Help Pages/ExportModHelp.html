Exporting the mod

Before exporting

Before exporting the mod, ensure that it has been saved at least once. Additionally, an export path needs to be set. It is advised to be C:\Users\**your username here**\Documents\my games\Terraria\tModLoader\ModSources as this is the location which tModLoader looks for mods. The path can be set in settings in the edit menu.

Exporting

The export button can be found in file menu. You will be given the opportunity to save the mod and increment the version number during the export process.

After exporting

After exporting the mod should appear in the develop mods section of tModLoader. This can be found by pressing the workshop button on the main menu. The new mod should say "rebuild required". Pressing "Build + Reload" will mean the mod is automatically added to the game. Due to an issue with tModLoader, it may not say "rebuild required". In this case, simply build the mod until "rebuild required" appears, then build it as normal.
